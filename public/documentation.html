Documentation Page

Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies

REQUESTS

Request URL HTTP Method Request Data Format RESPONSE DATA BODY FORMAT
Get a list of ALL movies to the user /movies GET none JSON Object holding data about all movies
Get description of movie /movies/:title GET none JSON Object holding data about specific movie
Get genre of movie /movies/genres/:genreName GET none JSON Object with movie according to param
Get directors info /movies/directors/:directorName GET none JSON Object of director
Get actors of movie /movies/:movieTitle/actors GET none JSON Object of starring actors
Get movies associated with actor /movies/actors/:actorName GET none JSON Object of all movies with same actor
Create new user /users POST A JSON Object holding information about user like this: { "name": "Mary", "favoriteMovies": "Inception"} A JSON Object holding information about user including ID like this: { "id": "1", "name": "Mary", "favoriteMovies": "Inception"}
Update user information(name) /users/:id PUT JSON Object containing name like :{ "name": "Mary" } JSON Object of updated user
Add movie to favorite movie list /users/:id/:movieTitle PUT none Text about outcome of request
Delete movie form favorite movie list /users/:id/:movieTitle DELETE none Text about outcome of request
Delete user /users/:id/ DELETE none Text about outcome of request
